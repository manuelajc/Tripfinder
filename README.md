## Tripfinder

## Descripocion del proyecto
Es una plataforma web diseñada para ser un intermediario entre turistas y traductores, proporcionando información detallada sobre diferentes destinoa turísticos, clasificados en entornos ruralesd y rurales. Ademas, facilita la conexion entre turistas y traductores disponibles en cada ubicacion.

## Caracterisitcas principales 
-Exploracion de lugares turisticos:Descubrir una variedad de destinos turisticos en entornos urbanos y rurales. 
-Selección de traductores: Los turistas pueden elegir una lista de traductores disponibles en cada ubicación.
